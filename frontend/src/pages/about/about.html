System Architecture

Mission Statement

CypherVault is a full-stack engineering project designed to demonstrate dual-state web architecture. It features a meticulously crafted UI mirroring elite cybersecurity platforms, powered by a Node.js/TypeScript backend.

Security Implementations

  • SQLi Defense: Parameterized queries via pg driver.
  • CSRF Mitigation: Custom Double-Submit Cryptographic Cookies.
  • SSRF Shielding: Internal IP (RFC 1918) range blocking.
  • XXE & XSS: Context-aware sanitization and DTD disabling.
